Release Notes - New in this Release

Release 7.50 rev 6 (April 9, 2019)

Enhancements and New Features
  • Support for New OS Versions:
    • Fedora 29
  • macOS only: The JDK to be used for starting SAP GUI for Java can now be defined by setting the path to the JDK bundle with the macOS user defaults system for domain com.sap.platin and key JDKPATH.
  • Tooltips: Showing the tooltip can be invoked by Ctrl+T.
  • Input History: Input History is available now also in editable Grid Control.
  • Keyboard navigation:
    • Keyboard navigation for RadioButtons in a RadioButtonGroup has been harmonised with the general keyboard navigation rules.
    • In Grid Control, support for Ctrl+Home, Ctrl+End, Ctrl+PgUP, Ctrl+PgDown has been added.
Bugfixes
  • High Contrast Theme:
    • Subtotals in Grid Control could not be read because of using wrong colors.
    • Some dialog windows did not have a white inset, so they have been hard to identify in front of black background.
  • Display of PDF based Content:
    • The new version of the third party component used for displaying PDF content introduced a bug causing the built-in print functionality not to work. A workaround has been implemented.
  • Keyboard navigation:
    • Opening the menu of a menu button in Toolbar Control by keyboard has not been possible. Now Alt-Cursor-down opens the menue.
    • When using Accessibility Mode, navigating to a disabled button, disabled radio button or disabled checkbox resulted in a dead end of tab chain.
    • Read-only radiobuttons in some cases could not be focused by keyboard in case of using Accessibility Mode.
    • While using Accessibility Mode, the focus traversal in Grid Control did not work correctly.
  • DropDown Listbox:
    • Flushing DropDown Listboxes, which have not been set to a default value, did not fire when choosing the empty item.
  • Grid Control:
    • Hitting the Escape key in editing mode reverts the edited value in a cell to the original value.
    • The feature for stretching text values when reporting changes has been missing, which has sometimes caused ABAP dumps.
    • Calls to SET_INFO_TABLE (setting properties ColPosIDOnDemand and SelectedColsOnDemand) could result in ABAP dumps in some cases.
    • Calls to DELETE_ALL_CELLS_BASE (deleteAllCells) could result in an exception.
    • A problem with the DELAYED_CHANGED_SEL_CALLBACK event has been solved.
  • Calendar Control
    • Various problems with selecting in interval and single day selection modes have been solved.
  • Inputfields
    • The F4-help buttons of inputfields, that are located at the very end of a group container, have not been visible. A workaround to horizontally extend the group container has been implemented.
  • Stability:
    • Various threading issues have been resolved in order to avoid deadlocks.
    • Some exceptions do no longer happen.
  • Windows only:
    • When setting the PLATIN_JVMOPT environment variable, an error in the start scripts caused the call to javaw command to fail.

Release 7.50 rev 5 (December 5, 2018)

Enhancements and New Features
  • Support for Java SE 11: SapMachine 11 with OpenJFX 11
  • Support for New OS Versions:
    • macOS Mojave (there are some known issues however)
    • SLED 15 and SLES 15
  • Grid Control:
    • Support for F2 keystroke to start cell editing has been implemented.
  • Grid Control and Calendar Control:
    • When using the animated focus indication, the then redundant dotted cell focus indication has been removed.
  • Display of PDF based Content:
    • A new version of the component to display PDFs has been added.
  • High Contrast Theme:
    • Windows and dialogs now have a thin white border. This is in particular helpful when running Dark Mode on macOS Mojave.
  • Icon font:
    • The icon font used in the themes Blue Crystal and High Contrast Black has been updated.
Bugfixes
  • SAP Screen Personas:
    • The proxy of components in Toolbar Control could not be found in case the function code of the component contains leading blanks.
    • An inconsistency setting maximum rows in Grid Control has been fixed.
    • A problem updating radiobuttons of a group from the backend has been solved.
    • The handling of pressF4() in Personas scripts on GuiTextFields has been incorrect.
    • Launching onBeforeRefresh scripts from GuiShellActions did not work.
  • Tree Control:
    • In case the width of the columns has not been set by the application, no content has been displayed. A workaround has been implemented.
    • The Tree Control has not been displayed in case setting the top node id was done before it has been created.
  • Grid Control:
    • A problem with the behavior of discontinuous selections has been solved.
  • Calendar Control
    • The focus navigation cycle with tab, shift-tab and focus group switch ctrl-tab did not work correctly.
  • Table Control:
    • The state of toggle buttons could have been wrong when scrolling vertically.
  • Tabstrip:
    • Screens containing a Tabstrip have been broken in case "Low speed connection" has been used.
  • Enhanced Search:
    • Typing has been blocked while waiting for the search results.
  • Keyboard:
    • Ctrl-F1 has not been reported to the backend.
  • Stability:
    • Various memory leaks related to the Html, Grid and Toolbar Control have been fixed.

Release 7.50 rev 4 (August 24, 2018)

Enhancements and New Features
  • Support for New OS Versions:
    • Fedora 28
    • openSUSE Leap 15
    • Ubuntu 18.04
  • SAP Screen Personas:
    • The Personas state now is visualized in the status bar with different icons and tooltip texts.
    • The new Personas icon in the status bar can be used to open the Personas Manager.
    • Support for ViewPort, which has been introduced with SAP Screen Personas 3.0 SP07, has been added.

    • The Scripting API resetProperty() and resetPropertyToOriginal() is now supported.
  • Modal dialogs:
    • Modal dialogs, which have been moved by the user, now keep their new position, even when they are temporarily deleted and recreated during a server round trip.
  • Icon font:
    • The icon font used in the themes Blue Crystal and High Contrast Black has been updated.
Bugfixes
  • Grid Control:
    • DropDown Listboxes set to read-only have been displayed with a wrong background color.
  • Html Control:
    • In some cases, an exception caused no content to be displayed.
  • Tree Control:
    • It could happen, that a node, that has been deleted in the meantime, is reported as current selection.
  • SAPInfo Control (ABAP class CL_GUI_FRONTEND_SERVICES):
    • Executing sapshcut and sapgui commands with relative path did not work.
  • SAP Screen Personas:
    • A Table Control moved to a different part of the screen could cause an exception after a round trip.
    • In Grid Control, an inconsistency with maximum rows and visible row count values has been solved.
  • Stability:
    • In special cases, it could happen that sessions were no longer responding.
    • A memory leak related to the Html Control and display of PDF content has been fixed.
  • Clipboard:
    • In some cases, pasting tab separated text had additional empty lines.
  • Localization:
    • Running with some locales resulted in failures during startup.
    • Enabling input history on a per field basis did not work in non-English locales.

Release 7.50 rev 3 (May 9, 2018)

Enhancements and New Features
  • SAP Screen Personas:
    • Flavors are only applied, when a theme is currently running that supports Personas.
    • When hiding the menus in a flavor, only the menus of the ABAP application are hidden to make sure other functionality like clipboard, tracing etc. still is availble.
    • In GuiTextField, the property recommended is now writeable in scripting.
    • In GuiToggleButton, the property selected has been introduced.
    • In GuiTableControl, when trying to apply selections in selectedColumns and selectedRowsAbsolute which are invalid, exceptions are thrown.
  • Grid Control:
    • Copying a range of rows and columns now is skipping empty rows and columns.
  • Desktop Integration:
    • Mime types for connection documents and SAP shortcuts (*.sapc, *.sap) have been added.
  • Misc:
    • Loading the central configuration using https is now supported.
Bugfixes
  • Table Control:
    • Various problems with layout calculation have been solved.
  • Grid Control:
    • There was an issue with typing AltGr characters into editable grid cells.
    • The font sizes of the cell renderer and editor have not been in sync.
    • Horizontal lines have unintentionally been shown in sorted columns in read only grids.
    • Various issues related to checkboxes in editable grid have been solved.
  • Tree Control:
    • The anchor selection was set to a wrong value with multi-selection and shift-click.
  • SAP Screen Personas:
    • Tab merging combined with highly dynamic sub screens resulted in wrong object locations and visibility.
    • Various issues in the Flavor Manager regarding rendering and search functionality have been solved.
    • When a Personas flavor has set the text value of a GuiTextComponent, the value change has not been reported to the back end.
    • The size of a page in a GuiTabStrip is now in sync with the size of the GuiTabStrip, which was modified by Personas.
  • Signature Design:
    • When using a system dependent scheme set by the user or back end, some background and text colors have been incorrect.
    • When setting the system dependent scheme to "Default" by the back end, always the blue scheme has been used.
    • The rendering of group boxes has been incorrect.
  • High Contrast Theme:
    • In tree control, the contrast of the text color of tree nodes and the selection background color has been improved.
  • Display of PDF based Content:
    • On some platforms, problems during initialisation of the component resulted in rendering issues.
  • Misc:
    • The overwrite caret has been rewritten to solve rendering issues in some environments.
    • Potential deadlocks related to controls have been solved.

Release 7.50 rev 2 (January 25, 2018)

Enhancements and New Features
  • Support for New OS Versions:
    • Fedora 27
    • openSUSE Leap 42.3
    • macOS High Sierra
  • SAP Screen Personas:
    • GuiGOSShell, GuiTitle and GuiStatusBar can now be used by SAP Screen Personas.
    • The functionality for tabToElement in GuiWrappingLabel has been implemented.
    • It is now possible to run onLoad or onAfterRefresh scripts during a tab-merge while a modal dialog on top is displayed.
    • GuiTextField now supports to run a onF4 script.
    • Modifying the maxLength of a Dynpro field is checked against the maxLength set by the application in order to avoid problems by exceeding this length.
    • In Tree Control, the double click event now can be invoked by scripting.
    • In Calendar Control, the tooltip property is now supported.
    • In Grid Control, the rowCount property is now supported.
  • High Contrast Theme:
    • In Grid Control, contrast of icons, focused cell background and colors for total row/column visualisation have been improved.
    • In Grid Control, while dragging an object the current drop target was barely visible.
    • In Table Control, the text in the header of selected columns now is easier to read.
    • In TextEdit Control, the rendering of borders and colors has been corrected. When being in read only mode, black background color instead of grey is used to increase the contrast.
    • The rendering of the sash in splitter and docker shells has been harmonised with the color palette of the High Contrast theme.
  • Color Schemes for Blue Crystal Theme:
    • The color scheme can now be set by the back end, please refer to SAP note 2572308.
  • Icon font:
    • The icon font used in the themes Blue Crystal and High Contrast Black has been updated.
  • Grid Control:
    • The selection functionality has been rewritten in order to make the behavior more consistent and intuitive.
    • The cell content layout is now more stable when entering and leaving a cell (switching between rendering and editing mode).
  • Window Menu:
    • The list of windows is now grouped by connections.
  • SAPInfo Control (ABAP class CL_GUI_FRONTEND_SERVICES):
    • DIRECTORY_LIST_FILES now reports additional file attributes.
  • Input History:
    • It is now possible to individually enable input history for fields with input length greater than 60.
Bugfixes
  • SAP Screen Personas:
    • Handling of styles related to read-only states has been corrected.
    • In DropDown Listbox, an issue handling the showKeys property has been solved.
    • In Grid Control, the column order has been incorrect when the column order was set by a flavor and the user manually reordered the columns.
    • A GuiTab in GuiTabStrip which has been disabled by a flavor, could still be activated by choosing the tab from the dropdown listbox.
    • In Table Control, the width property of columns has been handled with the wrong metric, rowCount and maxRows have been reported incorrectly and clearing the selection by clearSelection() did not work.
    • In Tree Control, various issues related to onSelectionChange() have been fixed.
    • It could happen, that controls modified by a flavor, disappear after a roundtrip.
    • In GuiTextFields, reporting the value of the recommended property has been incorrect.
    • In GuiToolbar, problems with reordering have been solved.
    • In GuiLabel, the iconID has not been reported in call cases.
  • Table Control:
    • The background color of headers has been wrong in various themes.
  • Enhanced Search:
    • In input fields part of a table control, enter and tab key for selecting an item did not work.
  • Control Enabling:
    • It could happen, that controls still have been visible, although the context has been changed.
  • Grid Control:
    • When extending the selection by ctrl-clicking on a cell, the cell has changed to edit mode.
    • In special cases, an icon with tooltip has not been displayed.
    • It has been possible to delete the content of a cell or column set to read only by keyboard, which resulted in an ABAP dump.
  • Group Container:
    • In case the text of a Group Container was not set, the position of its content has been wrong.
    • In case the frame has been hidden, the layout has been wrong.
  • Display of PDF based Content:
    • On some platforms, searching for the available fonts has failed resulting in rendering issues. A workaround for a bug in the third party component has been implemented.
  • Sashes:
    • In some environments, moving the sashes of Splitter and Docker did not work reliably and caused rendering problems. A workaround for a bug in the Java Runtime has been implemented and the bug has been reported to OpenJDK.
  • Chart Control:
    • A new version of the chart control is used to address various issues.
  • Rendering:
    • Unnecessary redraws have been removed in various components, which have been particularly annoying during window resizes involving server roundtrips.
  • Windows only:
    • In some environments, creating the Html Control failed because the Java Runtime was unable to load the required native DLLs. A workaround has been implemented.

Release 7.40 rev 9 (September 29, 2016)

Enhancements and New Features
  • Support for New OS Versions:
    • Fedora 24
    • macOS Sierra 10.12
  • DropDown Listbox:
    • Showing the keys on individual dropDown listboxes via Screen-Painter property independently from the local user settings is supported now.
Bugfixes

  • Shortcut file:

  • Security:
    • The dialog to change the trust level of a system failed to persist the chosen value in some languages.
  • OS X only:
    • A problem was fixed related to choosing the latest Java runtime on application launch.

    • The usage of a faster drawing API of Java introduced in 7.40 rev 3 has caused some artefacts in various environments depending on the OS resp. Linux distribution, graphics card and driver. A workaround has been implemented without falling back to the slower performance of previous releases.
  • Enhanced Search:
    • While using the enhanced search, it could happen that the same text typed before was filled into another input field as well.
  • Signature Design:
    • Highlighted text in some places was hard to read because of using a wrong foreground color when a scheme other then the default was used.
  • TextEdit Control:
    • The sequence of ABAP calls SET_TEXT_AS_R3TABLE, GET_TEXT_AS_R3TABLE and again SET_TEXT_AS_R3TABLE in DEFINE code for the control CL_GUI_TEXTEDIT resulted in displaying an empty text.
  • Tree Control:
    • In a special situation, an exception has caused the control to no longer repaint and react to user input.
  • Blue Crystal:
    • Checkmark icons in menues have not been displayed.
    • Menu separators have been missing.

Release 7.40 rev 3 (June 25, 2015)

Enhancements and New Features
  • Enhanced Search:
    • Support for limiting the search text to the field length has been implemented.
    • It is now possible to continuously type also while a search request is currently in process.
  • DropDown Listbox:
    • The feature to navigate and select an item by typing multiple characters has been implemented.
    • Navigating in the list using the keyboard is now consistent with SAP GUI for Windows.
  • Calendar Control:
    • Calendar Control now is based on the Date and Time API introduced in Java SE 8. This has not only greatly simplified the implementation but also added the possibility to support different calendaring systems.
  • SAPInfo Control (ABAP class CL_GUI_FRONTEND_SERVICES):
    • For Linux and OS X, the handling of paths has been made more robust regarding ABAP coding using the wrong path separator.
    • In method EXECUTE the parameter DEFAULT_DIRECTORY now is honored in case the document or application path is relative.
  • Display of PDF based Content:
    • A new version of the component to display PDFs is improving font management for some languages.
  • OS X only:
    • The rendering performance has been improved by using imaging API, that is hardware accelerated also on OS X. This also reduces resource consumption.
Bugfixes
  • Table Control:
    • The rendering of disabled line selection buttons has been corrected.
  • Html Control:
    • The query for the background color has returned a wrong value, which in some cases resulted in unreadable text.
    • When running embedded in Eclipse, problems with URLStreamHandlers resulted in showing empty areas instead of html content.
  • Tree Control:
    • An issue where calling CL_GUI_COLUMN_TREE->ADJUST_COLUMN_WIDTH has calculated an incorrect column width has been solved.
  • Display of PDF based Content:
    • Running with de-CH locale, the EURO sign has not been displayed.
